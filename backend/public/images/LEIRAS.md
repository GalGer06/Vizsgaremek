# Képek helye / Image Assets

Ide töltsd fel a képeket, amiket szeretnél használni a feladatokhoz vagy az oldalakhoz.
Upload the images here that you want to use for questions or pages.

### Elérési út / Path in code:
`http://localhost:3000/images/allomanynev.jpg`

### Példa használat a seederben / Example usage in seeder:
```typescript
{
  question: 'Mi látható a képen?',
  image: 'images/pelda.jpg', // Az adatbázisba így kerüljön
  ...
}
```
